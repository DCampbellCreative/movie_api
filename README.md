# FlixFix API

<p>A REST API created using JavaScript, Node.js, Express, Postman, and MongoDB.

Allows interaction with a hosted MongoDB database using CRUD operations.

Creates endpoints for routes to make requests to.

The Database contains an array of Movie objects containing details about each title and an array of User objects containing details about each user.

For more detailed info check out included documentation generated by JSDoc.
