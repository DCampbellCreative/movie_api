Movie API

This application provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request Body Data Response Body Data
Return a list of ALL movies to the user /movies GET A JSON object holding data about all movies.
Return data about a single movie by title to the user /movies/[title] GET A JSON object holding data about a single movie.
Return data about a genre by name/title /movies/[title]/[genre] GET A JSON object holding data about a movie's genre.
Return data about a director by name /movies/[title]/[director] GET A JSON object holding data about a director.
Allow new users to register /users POST A text message showing that the user's account was created.
Allow users to update their user info /users/[name] PUT A text message showing that the user's info was updated.
Allow users to add a movie to their list of favorites /users/[name]/favorites POST A text message showing that the favorited title was added.
Allow users to remove a movie from their list of favorites /users/[name]/favorites DELETE A text message showing that the favorited title was removed.
Allow existing users to deregister /users DELETE A text message showing that the user was deregistered.
